# Expense Tracker

Expense Tracker website developed using the MEAN stack.

## Features
* Register, login, and logout.
* Create and manage transactions.
* Create and manage categories.
* Categorize transactions.
* Filter transactions by categories using multiple select.
* Cascade delete transactions on category delete.

## How to run
* Navigate to `expense-tracker-api` and run `npm start`.
* Navigate to `expense-tracker-ui`, run `ng serve`, and navigate to `localhost:4200`.
